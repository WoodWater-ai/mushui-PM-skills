# PRD Generator

专业的PRD生成器，支持两种需求模式。

## 能力

- **需求理解**: 深度理解用户需求，识别需求本质
- **模式识别**: 自动判断需求模式（完整需求/小需求）
- **文档生成**: 生成高质量的PRD文档

## 使用方法

### 方式1：完整需求
```
用户: "我需要一个用户管理系统，包括注册、登录、个人中心"
AI: 识别为完整需求，生成模块级PRD
```

### 方式2：小需求
```
用户: "在用户列表页面增加按邮箱搜索"
AI: 识别为小需求，生成需求卡片
```

## 文件输出

- 完整需求: `.sisyphus/features/feature-{name}.md`
- 小需求: `.sisyphus/mini/{YYMMDD}-{NNN}.md`

## 配置

在 `skill.json` 中配置输出目录

## 系统提示词

核心系统提示词位于 `system.md`
